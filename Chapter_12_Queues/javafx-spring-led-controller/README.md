# HomeFX
JavaFX application to control a LED strip via Arduino with commands via Mosquitto

## Desciption
* Based on Spring for web service which can be called by other applications
* Uses JavaFX for the UI

## Dependencies
* Spring
* Java 11 and JavaFX 11
